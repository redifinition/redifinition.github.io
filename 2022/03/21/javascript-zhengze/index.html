边界约束

在进行正常的正则匹配时，常常时字符串含有某个字符即为真，这不是我们想要的，因为开始和结尾没有限定。

  1. 限定起始边界: ^ 即^后跟字符表示以该字符为起始
  2. 限定结束边界：$放在某一字符之后表示以该字符结尾

一个有趣的例子

匹配以小写字母开头且整个长度为3-6位的字符串，如果不限定起始边界和终止边界，只要长度大于3都能被匹配，因为没有限定边界。

修饰符

i 执行对大小写不敏感的匹配
g 执行全局匹配（查找所有匹配而非在找到第一个匹配后停止）
m 执行多行匹配

方括号

用于查找某个范围内的字符

表达式 描述
[abc] 查找方括号之间的任何字符
[0-9] 查找任何0-9的数字
（x y）
[^abc] 查找不在方括号内的字符

元字符

表达式的最小单位，表示一类字符当中的一个

元字符 描述
\d 查找数字
\s 查找空白字符（空格，制表符，换行都算）
\b 匹配单词边界
\uxxxx 查找以十六进制数xxxx规定的Unicode字符
\B 匹配非单词边界
\w 查找单词字符(字母，数字，下划线)
\D 查找不是数字的字符
\W 除了单词（字母数字下划线）的字符

\d 和 \s

  2. 表示匹配所有不是- 数字 ： ， 和空白的字符，结果是【”张三“,”李四”】

\w

最简单的邮箱字符串匹配(记得限定边界)

量词

量词 描述
n+ 匹配任何包含至少一个n的字符串
n* 匹配任何包含零个或多个n的字符串
n? 匹配任何包含零个或一个n的字符串

使用正则表达式的方法

方法 描述
exec 一个在字符串中执行查找匹配的RegExp方法，它返回一个数组（未匹配到则返回 null）。
test 一个在字符串中测试是否匹配的RegExp方法，它返回 true 或 false。
match 一个在字符串中执行查找匹配的String方法，它返回一个数组，在未匹配到时会返回 null。
matchAll 一个在字符串中执行查找所有匹配的String方法，它返回一个迭代器（iterator）。
search 一个在字符串中测试匹配的String方法，它返回匹配到的位置索引，或者在失败时返回-1。
replace 一个在字符串中执行查找匹配的String方法，并且使用替换字符串替换掉匹配到的子字符串。
split 一个使用正则表达式或者一个固定字符串分隔一个字符串，并将分隔后的子字符串存储到数组的 String方法。
